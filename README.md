# Yad2 Rent Watcher 📬🏠

This script monitors new or updated apartment rental listings from Yad2 and sends email alerts for:

- 🆕 New listings
- 💸 Price changes
- 🔁 Duplicate ads by same phone number

## 🚀 How to Use

### 1. Clone the repo:

```bash
git clone git clone git@github.com:liors16/alerts-yad2.git
cd alerts-yad2
```
